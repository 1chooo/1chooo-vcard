# Code Walkthrough

[1chooo.com](https://1chooo.com) is a web application that allows users to showcase their projects, blogs and portfolios. In the GitHub repository -- [1chooo.com](https://github.com/1chooo/1chooo.com), the codebase is organized into the mono-repo structure through Next.js Turbo. And we provide a high-level overview of the codebase and the architecture of the application. in this document.
